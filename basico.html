Selección del concepto: ¿Cómo elegimos el mejor diseño? Utilizando la Matriz de Pugh comparamos nuestras diferentes propuestas contra un diseño de referencia. Evaluamos cada opción basándonos en los requerimientos del usuario y criterios de ingeniería para elegir objetivamente el concepto ganador.

Análisis de Conceptos

Solución Verde

1. Dispensar medicamentos: El sistema rotatorio/botón permite ejecutar la entrega y, con apoyo del modo Manual, se asegura el control cuando se requiera intervención; los compartimentos se usan para almacenar medicamentos y también para separar dosis de forma ordenada.

2. Inventariar medicamentos: La base de datos registra existencias y movimientos para soportar avisar bajo inventario mediante mensaje, y activar alarma como medida inmediata para evitar medicamentos caducos.

3. Recordar tomar medicamentos: La cámara apoya la validación y seguimiento de la actividad del paciente, y con cada evento se genera un reporte que alimenta la bitácora e historial.

4. Progreso del paciente: La encuesta captura percepción y hábitos para detectar mal uso de medicamentos, se envía mensaje al médico cuando haya señales relevantes, y la pantalla muestra el aumento de información de dosis de forma clara para el paciente.

Solución Roja

1. Dispensar medicamentos: El modo Manual por emergencia permite activar el suministro cuando el sistema automático falla; el control de Tiempo dosis asegura que la entrega sea correcta y los Compartimentos organizan el almacenamiento y separación. En caso necesario, el modo manual permite intervenir directamente para garantizar continuidad del servicio.

2. Inventariar medicamentos: El led funciona como indicador visual del estado del inventario y los avisos notifican niveles bajos o próximos vencimientos; un segundo led puede reforzar alertas críticas para evitar errores operativos.

3. Recordar tomar medicamentos: El botón confirma la acción del paciente, se genera un mensaje de confirmación y se crea un reporte automático; además, otro reporte consolida el historial de cumplimiento para seguimiento clínico.

4. Progreso del paciente: El sistema puede emitir una receta médica basada en los datos recopilados, cerrando el ciclo entre monitoreo, registro y acción médica.

Solución Azul

1. Dispensar medicamentos: El modelo híbrido combina supervisión humana con operación automática, permitiendo que el sistema funcione solo pero con posibilidad de intervención cuando sea necesario.

2. Inventariar medicamentos: La información se gestiona en excel/programa, donde se registran existencias, movimientos y patrones de consumo para mantener control estructurado.

3. Recordar tomar medicamentos: Un sensor detecta interacción o presencia; si no se cumple la toma en el tiempo esperado, se activa una alarma y se envía un mensaje como recordatorio inmediato.

4. Progreso del paciente: El sistema genera una alerta cuando detecta comportamientos irregulares o riesgo clínico, permitiendo escalar la situación de manera oportuna.
